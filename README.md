# Catter
DBMT的Blender插件

# 开发环境
- Visual Studio Code
- Python 3.12.7
- Blender 4.2 LTS

# 注意事项

- 文件夹尽量小写，否则VSCode无法追踪文件夹大小写改变的git记录。
- 单元测试写到unity_test文件夹下，不要在类里直接定义__main__函数执行。
- 属性统一放到config/catter_properties下，全局统一管理
